# jQuery Learning Journey

A collection of self-contained HTML examples demonstrating core jQuery concepts. Each file focuses on one concept, organized under `src/`.

## Topics Covered

### Basics & Effects
- **src/index.html** - jQuery setup, click/mouse events, effects (toggle, fade, slide, animate), DOM manipulation, CSS methods

### DOM Traversing
- **src/01-dom-traversing/index.html** - Traversing the DOM with `.closest()`

### Event Delegation
- **src/02-event-delegation/index.html** - Handling events on dynamically created elements using `.on()`

### AJAX
- **src/03-ajax/index.html** - Making HTTP requests with `.ajax()`

### Form Serialization
- **src/04-form-serialize/index.html** - Serializing form data with `.serialize()`

### Attributes & Data
- **src/05-attributes-data/index.html** - Getting/setting attributes with `.attr()` and `.data()`

### Iteration & Chaining
- **src/06-each-chaining/index.html** - Iterating with `.each()` and method chaining

### Deferred / Promises
- **src/07-jquery-deferred/index.html** - Using `.done()` and `.fail()` with AJAX promises

### Plugin Creation
- **src/08-plugin-example/index.html** - Extending jQuery with custom plugins via `$.fn`
